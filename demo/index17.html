管道参数
slice 截取 格式 data |slice:start:end 不要把参数用引号包起来 {{ '01234567890' | slice:1:4 }}
number 2.2-2 保留小数(要处理的数字不要用引号，不然会被当成字符串，它只能对数字进行操作) {{avgScore() | number:'2.2-2'}}
data yyyyMMYDD 格式化 时间格式具体去baidu {{ day | data:'yyMMdd'}}
json 转成json {{ {x:1,y:2} | json }}
percent 转换成百分比 {{ 1.23456 | percent:'1.2-3' }}
uppercase 大写转换 {{ "this is a demo" | uppercase }}
lowercase 小写转换 {{ "WHAT A WONDERFUL WORLD" | lowercase }}
